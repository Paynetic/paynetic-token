# Paynetic Token (PNT)

PNT is an Ethereum ERC20 token, used throughout the Paynetic ecosystem.

## Usage

### Format and Test

```shell
$ forge fmt
$ forge test
```

### Build

```shell
$ forge build
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ forge script script/PNTToken.s.sol:PNTTokenScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
